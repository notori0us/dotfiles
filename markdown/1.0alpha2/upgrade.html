Upgrading from 1.0alpha1 "Appa" to 1.0alpha2 "Momo"

Upgrading an installation from a prior version is not officially tested or supported. Attempt to do so at your own risk.

If you would like to attempt to upgrade despite the above warning, download the userland tarball into a temporary directory and replace core Bedrock files listed below with the ones from the tarball:

  • /etc/init.d/rcK
  • /etc/init.d/rcS
  • /etc/profile
  • /opt/bedrock/bin/brc
  • /opt/bedrock/bin/brl
  • /opt/bedrock/bin/brw
  • /opt/bedrock/sbin/brp
  • /opt/bedrock/sbin/bru

Moreover, you should compare the following files with those on your system and use your own judgement about necessary changes:

  • /etc/fstab (note the comment improvements)
  • /etc/profile (note the change from TIMEZONE to TZ)
  • /etc/rc.local (note the comment improvements)
  • /opt/bedrock/etc/brclients.conf (note the comment improvements)
  • /opt/bedrock/etc/rc.conf (note the change from TIMEZONE to TZ, addition of UDEV_CLIENT)

Finally, you should run 

brp -a
